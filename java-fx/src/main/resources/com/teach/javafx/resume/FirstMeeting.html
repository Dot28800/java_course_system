2.简要的任务说明

新闻稿部分（注意字数大约600左右）

任务一、任务五：都是对于会议内容的报道，图片可以截腾讯会议截面图以及会议所分析内容（推荐），文字推荐概括会议主要内容（后续这个文档会发的不用急），相对简单一些

任务二三四：是具体旅游期间的报道，聚焦于不同方面。任务二集中于西安的厚重历史文化底蕴，在叙述游踪基础上可以尝试加入团队成员的感受（但不要以第一人称写）；任务三聚焦于现代西安的发展状况，难度相对较大，游览过程中除任务二的部分都可以考虑写（主要是展现外乡人眼中西安的城市印象）；任务四以延安为主体，可以侧重展现对革命文化的追忆和对新时代青年的作用，也可说明自己对延安精神的理解。 以上均是本人建议，具体写什么由各位自行决定，只要不是过于超凡脱俗就行。

稿件内时间一律按分工文件上来即可，如任务二稿件中的时间写2024年2月21日，并且由于审核要求的时间与稿件时间相差两天以内否则不予通过，希望各位能尽早完成（次日一定要上传了）上传到群文件里面的样式可以参考学长学姐们的威海刘公岛那个。

字体样式、大小等参照分工文档中的要求，只要写好文稿附带上图片打包就行不用专门排版。

照片要求：1.照片需要反应活动的现场场景，比如发放问卷、调研现场等，不要只有团队合照或景观；2.出镜人员姿势端庄，没有睡觉、玩手机等情况

注意点：1.新闻稿的真实性和时效性；2.准确性：（1）无错别字（2）出现的专有名词、引用的话、对某个人的介绍等内容确保准确（3）涉及政治性的词语一定一定一定注意标准表述 ；3.客观性：避免使用“我”“我们”“圆满结束”等主观性的词语；避免新闻稿中出现带有个人观点的语句。
